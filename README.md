# Numerov Schrödinger Solver

**Author:** Alon Sportes  

This project solves the 1D time-independent Schrödinger equation using the Numerov method combined with shooting and root finding.

It computes bound states for several potentials, validates against analytic solutions, performs convergence studies, and explores tunneling and scattering phenomena.

## Installation

bash pip install -r requirements.txt 

## Run the project

bash python scripts/run_solver.py 

## Output

Results are saved in:

results/

including:
- wavefunctions and spectra
- convergence plots
- tunneling and scattering results
